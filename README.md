# bingo-buda-backend

## Description

Proyecto de prueba tecnica para Sofka U ⚡️

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
